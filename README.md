# spog-the-dog

Spog is a 4 legged robot leg modelled after Boston Dynamic's Spot. 

Spog can:
- Walk forward, backward
- Turn left and right
- Forward object detection
- Dance (if you call stepping forward and backwards repeatedly a dance)
